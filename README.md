# Set up the dev env

Bower :
```
npm install -g bower
```

Yeoman :
```
npm install -g yo
```

Ionic & Cordova :
```
npm install -g cordova
npm install -g ionic
npm install -g generator-ionic
```

# Create the base app

```
mkdir whatsapp
cd whatsapp
yo ionic whatsapp
```

# Install dev dependencies

```
npm install
```

# Install dependencies

```
bower install
```

# Start the server

```
grunt serve
```

# Application structure

- /app : the application
- /wwww : generated application
- /node_modules
